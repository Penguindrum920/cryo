Place each scroll-scrubbed JPG sequence in one of these folders:

- Frostbite Berry frames
- Neon Meltdown frames
- Cosmic Crush frames
- Midnight Citrus frames
- Velvet Frost frames

The app sorts frame filenames naturally, so names like frame_0001.jpg,
frame_0002.jpg, and frame_0100.jpg will scrub in order.
